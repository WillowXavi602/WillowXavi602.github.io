    Carrot vs Spudnik

    It was my first game and was a highly detailed little platformer about a veggie pirate chasing down a potato dog and was fun to put together. I learned how to make a platformer and many animations through this project. I also learned how to use Unity and how to make a game from start to finish. it really tested my art and coding skills to create and present to the teacher and im still proud of it to this day. I hope to make more games like this in the future.

    skills i had to use on this include, coding, art, and animation. I had to learn how to use Unity and how to make a game from start to finish.
